## 明确偏离原项目的决策

1. **LPMM 知识图谱（KG/openIE/quick_algo）未迁移**——原项目该链路重度依赖私有 quick_algo 与批量导入管线，且群聊场景实际命中率低。现有三层记忆（摘要+faiss+画像）覆盖其主要价值。如需知识库问答，建议后续以 MCP server 形式外挂。
2. **WebUI 前端重写**——旧 dist 与旧 API 深度耦合（多 bot 管理/git 镜像等已砍功能占一半路由），轻量单页反而完整覆盖验收项。
3. **hippo 摘要简化**——原话题缓存累积更新机制简化为批次触发，语义等价（压缩上下文→长期记忆）。
4. **expression reflector 未做**——向管理员求证表达的功能使用频率极低，接口预留。
